# visual-platform
光迹 - 视觉服务可信交易平台，基于 Spring Boot+Vue 开发的全栈 Web 项目，聚焦视觉服务供需双方交易场景，实现作品分享、需求撮合、交易履约保障、双向互评、数据统计分析全流程功能，打造安全可信的视觉服务交易生态，支持摄影师、需求方、平台管理员多角色协同操作。
## 技术栈
### 后端技术栈
- 核心框架：Spring Boot 2.x / Spring MVC / Spring Security
- 持久层：MyBatis-Plus / MySQL 8.0
- 工具组件：Redis（缓存）、JWT（身份认证）、Lombok
- 部署运维：Maven、Git、Linux
### 前端技术栈
- 核心框架：Vue 2 / Vue CLI
- UI 组件：Element UI
- 网络请求：Axios
- 路由状态：Vue Router、Vuex
### 全栈架构
- 前后端分离架构，RESTful API 接口规范，支持多角色权限管理、高并发缓存优化、交易数据安全存储。
